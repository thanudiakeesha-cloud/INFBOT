User Manual

🚀 Getting Started

  1. Go to Deploy Bot from the sidebar
  2. Choose: Session ID, Pair Code, or QR Code
  3. Bot appears in Bot Sessions once connected
  4. Customize behavior in Settings
  5. Browse all commands in Commands

⌨️ Key Commands

.menu — Main menu
.settings — Bot settings
.mode — Private/Public mode
.alive — Check bot status
.sticker — Create stickers
.ai — AI chat

🔗 Connection Methods

  • Session ID — Paste your session string
  • Pair Code — Enter number, link in WhatsApp
  • QR Code — Scan from WhatsApp Linked Devices
